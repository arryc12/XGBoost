# XGBoost + SHAP 流型分类模块

本模块使用 XGBoost 进行气液两相流流型分类，并结合 SHAP 进行特征重要性分析，同时使用麻雀搜索算法 (SSA) 自动优化模型超参数。

## 功能特点

- **自动超参数优化**：使用 SSA 算法自动搜索最优参数组合
- **流型分类**：基于 XGBoost 的多分类模型
- **特征解释**：SHAP 分析特征对预测结果的影响
- **类别不平衡处理**：自动计算类别权重
- **优化过程可视化**：生成优化曲线图

## 使用方法

### 1. 准备数据集

将数据集放在 `datasets/` 目录下，CSV 格式，最后一列必须为 `label`。

参考格式 (`annular.csv`)：
```
feature1,feature2,...,featureN,label
0.123,0.456,...,0.789,0
0.234,0.567,...,0.890,1
```

### 2. 修改数据路径

在 `test.py` 中修改数据加载路径：

```python
df = pd.read_csv(r'XGBoost_SHAP\datasets\your_data.csv')
```

### 3. 运行脚本

```bash
cd XGBoost_SHAP
python test.py
```

## SSA 优化参数说明

| 参数 | 默认范围 | 说明 |
|------|----------|------|
| 树的数量 | 100-500 | 决策树总数 |
| 树的最大深度 | 3-15 | 控制模型复杂度 |
| 学习率 | 0.01-0.2 | 每棵树的贡献权重 |
| 样本采样比例 | 0.6-1.0 | 防止过拟合 |
| 特征采样比例 | 0.6-1.0 | 增加随机性 |
| 子节点最小权重 | 1-10 | 保守分裂阈值 |
| 分裂最小损失增益 | 0-1 | 节点分裂条件 |
| L1正则化系数 | 0-1 | 特征选择 |
| L2正则化系数 | 0-1 | 权重平滑 |

## 输出文件

| 文件 | 说明 |
|------|------|
| `ssa_optimization.png` | SSA 优化过程曲线 |
| SHAP 图 | 特征重要性分析图 |

## 注意事项

1. SSA 优化需要较长时间，可通过调整 `n_sparrow` 和 `n_iter` 控制优化时间
2. 数据集标签列必须命名为 `label`
3. 优化过程中会打印每 10 次迭代的当前最优准确率
